FreeCodeCamp API Basejump: Timestamp Microservice

User Stories

  • I can pass a unix timestamp or a natural date string as a parameter, and it will be returned along with the other type
  • If the parameter does not contain a timestamp or date string, null will be returned for both properties

Usage Example

https://timestamp-ms.herokuapp.com/December%2015,%202015
https://timestamp-ms.herokuapp.com/1450137600

Response Example

{ "unix": 1450137600, "natural": "December 15, 2015" }
